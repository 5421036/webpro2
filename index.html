・The img element

文書内に画像を表示するためのタグです。classで名前をつけ、cssを利用することで、画像の大きさを変えられる。表示する画像ファイルはsrc属性で指定する。

マークアップ例
<img class="image"src="images/mydog.jpg"alt="dog">
実行結果

・The ul element

順序なしリストとのことを指し、一般的に、行頭記号を伴うリストとして描画されます。

マークアップ例
【今ほしい物リスト】
<ul>
<li>大きいテレビ</li>
<li>犬</li>
<li>Nintendo Switchのソフト</li>
<ul>
<li>テラリア</li>
<li>モノポリー</li>
<li>マリオカート</li>
</ul>
<li>床暖房</li>
</ul>
実行結果
【今ほしい物リスト】
  • 大きいテレビ
  • 犬
  • Nintendo Switchのソフト
    • テラリア
    • モノポリー
    • マリオカート
  • 床暖房

・The h1, h2, h3, h4, h5, and h6 elements

HTML の h1,h2,h3,h4,h5,h6要素は、セクションの見出しを6段階で表します。h1 が最上位で、h6 が最下位です。

マークアップ例
<h1>見出し１</h1>
<h2>見出し２</h2>
<h3>見出し３</h3>
<h4>見出し４</h4>
<h5>見出し5</h5>
<h6>見出し６</h6>
実行結果

見出し１

見出し２

見出し３

見出し４

見出し5
見出し６
